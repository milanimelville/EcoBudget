EcoBudget is a dynamic Django based web application that helps users track their spending while encouraging environmentally conscious habits. 
Users can log their transactions, view their spending history and monitor eco-friendly vs. non eco-friendly expenses through their EcoBudget dashboard.

# Features
- User authentication
- Expense Dashboard
- Add and view expenses
- Categorise purchases as eco-friendly or non eco-friendly
- Spending summary

# Technologies
- Python 3
- Django
- SQLite development database
- HTML
- CSS
- Javascript
- Render

# Link to the Website
https://ecobudget.onrender.com/
This project requires a backend server so it was deployed using Render.

# How to use
1. Open the live link above.
2. Register a new account and log in.
3. You will be directed to the Eco dashboard where you can view spending summaries, add purchases and view purchase history
